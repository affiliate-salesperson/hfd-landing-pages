# HFD Landing Pages — Health Flow Daily™  

This repository contains all landing pages for the Health Flow Daily™ ecosystem:

- NSR™ (Nervous System Reset)
- Sleep Reset 28™
- Uplift Day™
- Future premium products

All pages follow a strict minimalist, premium aesthetic and are built with:

- Pure HTML + CSS  
or  
- HTML + Tailwind (if enabled)

No copy should ever be altered inside this repository. Text is final and provided externally.

## Structure

/nsr  
/sleep-reset  
/uplift-day  
/brand  

Each folder contains:

- index.html  
- style.css  
- optional components  

## Deploy

This repo is connected to Vercel.  
Each subfolder deploys to a different project.

## Important

No random files, no scripts, no frameworks unless explicitly requested.
